# 🛍️ Fusion Mart | Smart Shopping Experience

A modern, fully responsive e-commerce web application with dark/light theme, shopping cart, wishlist, and advanced product filtering.
Built with pure HTML, CSS, and JavaScript.

## 🌐 Live Demo

**Online Demo:** [https://mohammad-2012.github.io/Fusion-Mart/](https://mohammad-2012.github.io/Fusion-Mart/)

---

## 📝 Description

Fusion Mart is a complete shopping cart application that allows users to browse products, add items to cart, save favorites to wishlist, 
search products by name, filter by categories, and toggle between dark/light themes. The app fetches real product data from FakeStore API and stores
cart/wishlist data in localStorage. Fully responsive design works perfectly on desktop, tablet, 
and mobile devices with bottom navigation bar for mobile users.

---

## ✨ Features

### Core Features
- **Product Listing** — Grid layout with product images, titles, prices
- **Search Products** — Real-time search by product name
- **Category Filtering** — Filter products by electronics, jewelry, clothing
- **Shopping Cart** — Add/remove items, increase/decrease quantity
- **Wishlist** — Save favorite products with heart button
- **Dark/Light Theme** — One-click theme toggle with localStorage persistence
- **Responsive Design** — Optimized for all screen sizes

### Additional Features
- Local storage persistence for cart and wishlist
- SweetAlert2 notifications for user actions
- Keyboard support (optional)
- Loading skeletons with spinner animation
- Sticky header with search and theme toggle
